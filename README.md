# Movies App

Aplicación que permite buscar tus películas y series favoritas.

![app.gif][1]

## Tecnologías
* React
* React-router
* Webpack
* Fetch
* OMDb API

## Instalación y Ejecución
```shell
  $ git clone https://github.com/jorcalle11/movies-app-react.git
  $ cd movies-app-react
  $ npm install
  $ npm start
```
